This is a collection of animated GIFs to be used with the AnimatedGIFs and the LightAppliance Teensy 3.1 sketches.

These five directories of GIFs must be subdirectories of the root directory on the SD card.

The 4thgifs directory contains 4th of July animations
The gengifs directory contains animations of a general nature
The halogifs directory contains halloween animations
The valgifs directory contains valentine animations
The xmasgifs directory contains Christmas animations
